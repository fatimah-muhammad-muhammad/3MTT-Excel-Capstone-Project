# Sales Performance Dashboard

## Project Overview
This repository contains a dynamic **Sales Dashboard** built in Microsoft Excel. The project was designed to transform raw sales data into a visual, interactive tool for monitoring business health and performance metrics.

## Features
* **Interactive Slicers**: Filter data instantly by Month, Store, Salesman, or Item.
* **Sales by Product**: A pie chart visualizing the revenue distribution across items like Televisions, Home Theaters, and Cell Phones.
* **Total Sales Trends**: A bar chart tracking monthly performance.
* **Sales by Store & Personnel**: Comparative bar charts to identify top-performing locations and team members.

## Technical Skills Demonstrated
* Data analysis and cleaning.
* Pivot Tables & Pivot Charts.
* Dashboard UI/UX design in Excel.
